# SOC Alert Triage Training

This project documents a beginner-friendly SOC training resource focused on alert triage fundamentals, workflow, classification, documentation, and analyst decision-making.

## Project Overview

This training package was created as part of my SOC portfolio to demonstrate understanding of how a Tier 1 analyst reviews, validates, classifies, and documents security alerts.

The guide covers:
- Introduction to alert triage
- Common SOC alert types
- A 7-step triage workflow
- Core log sources used during investigation
- Alert classification logic
- A brute force investigation case study
- A triage report template
- Common beginner mistakes
- A summary checklist

## Project Files

- `SOC_Alert_Triage_Guide.pdf` — Full instructional guide
- `case-study-summary.md` — Short project case study summary
- `triage-report-template.md` — Reusable triage report template
- `workflow-checklist.md` — Quick triage checklist for investigations

## Skills Demonstrated
