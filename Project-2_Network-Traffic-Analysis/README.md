# Project 2: Network Traffic Analysis

## Objective
To analyze network traffic from a SOC analyst perspective using PCAPs and network security tools in order to identify suspicious and malicious behavior.

## Skills Demonstrated
- Network traffic analysis (DNS, HTTP, TLS)
- PCAP investigation
- SOC-style evidence collection
- IOC identification
- Documentation and reporting

## Tools Used
- Wireshark
- Zeek
- Suricata
- VirtualBox Lab Environment

## Project Structure
- evidence/ → screenshots, pcaps, logs, investigation notes
- iocs/ → identified indicators
- report/ → final incident-style report
